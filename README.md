Общее описание программы
Программа представляет собой графическое приложение (GUI) на Python для генерации случайных цитат. Она позволяет:

Генерировать случайную цитату из встроенного списка

Сохранять все сгенерированные цитаты в историю

Фильтровать историю по автору и теме

Добавлять новые цитаты через интерфейс

Сохранять историю в JSON-файл для сохранения между запусками

Используемые библиотеки
tkinter – для создания графического интерфейса

random – для случайного выбора цитаты

json – для сохранения и загрузки истории

datetime – для добавления временной метки к цитате

Структура программы
1. Класс QuoteGenerator
Основной класс, управляющий всем приложением.

Атрибуты:

quotes – список предопределённых цитат (каждая цитата – словарь с ключами: text, author, topic)

history – список сгенерированных цитат + временные метки

root – главное окно приложения

Методы:

__init__ – инициализация: создание окна, загрузка истории, создание интерфейса

create_widgets – создаёт все элементы GUI (кнопки, поля, списки)

generate_quote – выбирает случайную цитату, добавляет в историю, сохраняет в файл

update_history_display – обновляет список истории с учётом текущих фильтров

filter_history – применяет фильтрацию (вызывается при выборе фильтра)

reset_filters – очищает фильтры

add_quote – добавляет новую цитату с проверкой на пустые строки

save_history / load_history – сохранение/загрузка истории в JSON

2. Графический интерфейс
Интерфейс разделён на три логические области:

Верхняя область (Генератор цитат)

Кнопка «Сгенерировать цитату»

Метка для отображения текущей цитаты

Средняя область (Фильтрация)

Выпадающие списки для выбора автора и темы

Кнопка «Сбросить фильтры»

Кнопка «Добавить цитату»

Нижняя область (История)

Список всех сгенерированных цитат с временными метками

Полоса прокрутки

3. Проверка корректности ввода
При добавлении новой цитаты (add_quote) программа проверяет:

Текст цитаты не пустой и не состоит из пробелов

Имя автора не пустое

Тема не пустая

Если какое-то поле пустое, появляется предупреждение.

4. Сохранение данных
Все сгенерированные цитаты сохраняются в файл quote_history.json в формате:

json
[
  {
    "text": "Текст цитаты",
    "author": "Автор",
    "topic": "Тема",
    "timestamp": "2026-05-07 14:30:00"
  }
]
При запуске программа автоматически загружает историю из этого файла.
